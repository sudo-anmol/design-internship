## 4. A project you've shipped recentlyI recently built and shipped CivicLens, a full-stack civic complaint platform that allows users to report and track local issues with location and image support.

The system was built using Next.js, TypeScript, Supabase, and Google Maps API. I implemented role-based dashboards for citizens, authorities, and admins, along with secure authentication and real-time status tracking for complaints. The platform supports over 100+ reported issues with structured workflows for updates and resolution tracking.

One of the main challenges was designing a clean data model and ensuring smooth state management across different user roles. I also focused on performance and usability, making sure the platform remained responsive while handling multiple reports and updates.

This project helped me understand how to build and deploy a complete product—from backend logic and database design to frontend experience—while solving a real-world problem.
