5. A project you've shipped recently
https://diya777-coder.github.io/
LifeOS 2035 is a speculative critical design project that imagines a fictional life operating system which scores every dimension of human existence including health, emotion, relationships, career, and finances into a single algorithmic Life Score. I designed the full UI and system logic and vibe coded the prototype, building out the interface, the scoring dashboard, and a day-in-the-life narrative that shows exactly where the system helps and where it starts to control you. If I were to revisit it, I'd push the onboarding flow harder so the user feels genuinely excited about LifeOS before the unsettling parts land, because the discomfort hits much harder when you've already bought in.
