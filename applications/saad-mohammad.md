## 4. A project you've shipped recently

**ArchAI — AI System Architecture Generator**
https://portfolio-saadonwebs-projects.vercel.app/archai-landing.html

ArchAI is a tool where you describe an app idea in plain English and get a complete, live system architecture diagram in seconds — powered by Google Gemini and rendered with Mermaid.js. I built it end-to-end: the prompt engineering, the streaming API integration, the diagram renderer, and the UI. What I'd change now: make it conversational, so you can iterate on the architecture through follow-up prompts instead of regenerating from scratch each time. That feels like the obvious next version.
