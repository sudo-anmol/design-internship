## 4. A project you've shipped recently

One project I recently built is an engine spare parts management and ordering website for a vehicle parts company. The platform allows users to browse spare parts, place orders, and generate invoices, while the admin dashboard manages products and orders.

I worked on the frontend UI, database structure, and backend functionality using HTML, CSS, JavaScript, PHP, and MySQL. The biggest challenge was designing a simple interface for mechanics and garage owners who are not used to complex digital systems.

If I were to improve it now, I would optimize performance further and rebuild parts of the UI using a modern framework like React or Next.js for better scalability.
