## 4. A project you've shipped recently

_Most recently I built my personal lab {https://piyush-lab.vercel.app/}page — a creative portfolio experiment using Google Antigravity and Claude to vibe-code the entire thing from scratch. I had only a basic understanding of HTML and CSS, so the process was very hands-on: I'd describe what I wanted, get code back, run it in the browser, see what broke, and iterate. The biggest challenge was performance — the page was loading slowly because of heavy canvas elements and p5.js interactions, so I went back through the code, optimised the rendering loops, and cut down unnecessary redraws until it felt fast and smooth.

I used p5.js to build interactive generative elements things that respond to cursor movement and bring the page to life — which I wouldn't have been able to do with just Figma. The whole thing was built with design and production running hand in hand: I wasn't designing first and coding later, I was watching it come to life in the browser and adjusting both the visual direction and the code at the same time. It changed how I think about the line between design and engineering.
